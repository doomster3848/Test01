제어
좌측 패널은 스크롤을 따라 고정됩니다. ROI는 원본 기준 또는 전체 패널 기준으로 선택할 수 있습니다.

PNG / JPG / BMP / WEBP
공통 표시/ROI 제어 설정
현재 설정만 저장합니다. 이미지는 저장하지 않습니다. 길이 측정 ON 상태에서 원본 화면을 두 번 클릭하면 거리(px)가 표시됩니다.
Kernel Filter
3×3 프리셋 또는 5×5 수동 입력으로 Kernel을 적용합니다.
Low-pass: Blur / Gaussian · High-pass: Sobel / Laplacian / Sharpen
Kernel 값은 정수로 입력하고, 1/9 · 1/16 같은 나누기 값은 Offset에 그대로 입력하세요. 음수(-)는 주변 값을 빼서 경계를 강조합니다.
안전 모드: 다중 Kernel은 아래 Kernel 결과 캔버스에만 적용됩니다. 원본 보기, 검출, Blob 계산은 기존 단일 Kernel 기준을 유지합니다.
